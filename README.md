# MALKU - Sitio Web Estático de Ejemplo

## 📖 Descripción

**Malku** es un proyecto académico desarrollado como ejemplo de organización y maquetación de un sitio web estático utilizando únicamente **HTML5 y CSS3**.

El objetivo de este repositorio no es implementar funcionalidades dinámicas ni lógica de negocio, sino servir como referencia para que los estudiantes puedan observar:

* Cómo estructurar un proyecto web.
* Cómo separar los estilos por páginas.
* Cómo reutilizar estilos globales.
* Cómo organizar carpetas y archivos.
* Cómo diseñar interfaces modernas y responsivas.
* Buenas prácticas básicas de maquetación.

Este proyecto simula una tienda de indumentaria de estilo minimalista y editorial, inspirada en marcas de diseño contemporáneo.

---

## 🎯 Objetivos Educativos

Este repositorio fue creado para que los alumnos puedan aprender:

* Estructura básica de un sitio web.
* Organización modular de archivos CSS.
* Uso de clases e identificadores.
* Diseño responsive mediante Media Queries.
* Uso de variables CSS.
* Aplicación de Flexbox y CSS Grid.
* Separación de responsabilidades entre contenido (HTML) y presentación (CSS).
